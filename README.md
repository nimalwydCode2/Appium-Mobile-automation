# Mobile Automation Framework — Appium + Java + TestNG

A reusable, scalable and modular mobile automation framework built using:
- Java
- Appium (W3C Actions)
- TestNG
- Page Object Model (POM)

## Features
✔ Android & iOS capability builder  
✔ W3C gestures: swipe, scroll, tap, long press  
✔ Modular utilities for gestures & waits  
✔ Reusable DriverFactory  
✔ Page Object Model (POM)  
✔ Cross-platform support  
✔ CI-ready structure  

## Folder Structure
src/main/java
│── base
